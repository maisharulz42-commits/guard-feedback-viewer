# Guard Training Feedback Viewer

A web application to view guard training feedback data from Google Sheets.

## Features
- Real-time data from Google Sheets
- Search and filter functionality
- Statistics dashboard
- Export to CSV
- Mobile responsive design

## How to Use
1. Click "Load Data from Google Sheets"
2. Use search and filters to find specific feedback
3. Click "View" to see detailed ratings
4. Export data using the "Export CSV" button
